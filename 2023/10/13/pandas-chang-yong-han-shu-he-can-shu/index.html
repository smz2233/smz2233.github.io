2. 写CSV文件

DataFrame.to_csv(path, index=True, encoding=None)

  • path: 表明文件系统位置的字符串，必选参数

  • index: 是否保存索引，默认 index=True

    • 使用示例

      >>> import pandas as pd
>>> df = pd.DataFrame(np.arange(9).reshape((3, 3)),
                      index=['a', 'b', 'c'], 
                      columns=['Ohio', 'Texas',
                               'California'])
>>> df
   Ohio  Texas  California
a     0      1           2
b     3      4           5
c     6      7           8

>>> df.to_csv('test.csv')
>>> pd.read_csv('test.csv')
  Unnamed: 0  Ohio  Texas  California
0          a     0      1           2
1          b     3      4           5
2          c     6      7           8

>>> df.to_csv('test.csv', index=False)
>>> pd.read_csv('test.csv')
   Ohio  Texas  California
0     0      1           2
1     3      4           5
2     6      7           8

  • encoding: Unicode文本编码，默认 encoding=’utf-8‘，含有中文文本建议修改参数值 encoding=‘gbk’

二、轴向上删除条目

DataFrame.drop(labels=None, axis=0, inplace=False)

  • labels: 要删除的标签序列

  • axis: 删除的轴向，默认 axis=0(行标签)

  • inplace: 修改被调用的对象，而不是生成一个备份， 默认 inplace=False(不修改被调用的对象)

    • 使用示例

      >>> import numpy as np
>>> import pandas as pd
>>> df = pd.DataFrame(np.arange(9).reshape((3, 3)),
                      index=['a', 'b', 'c'], 
                      columns=['Ohio', 'Texas',
                               'California'])
>>> df
   Ohio  Texas  California
a     0      1           2
b     3      4           5
c     6      7           8

# 删除行标签
>>> df.drop('a')
   Ohio  Texas  California
b     3      4           5
c     6      7           8
>>> df.drop(['a', 'c'])
   Ohio  Texas  California
b     3      4           5

# 删除列标签
>>> df.drop('Ohio', axis=1)
   Texas  California
a      1           2
b      4           5
c      7           8
>>> df.drop(['Ohio', 'California'], axis=1)
   Texas
a      1
b      4
c      7

# 修改被调用的对象, 注意应用此方法没有返回值
>>> df.drop('a', inplace=True)
>>> df
   Ohio  Texas  California
b     3      4           5
c     6      7           8
>>> df.drop(['Ohio', 'California'], axis=1,
            inplace=True)
>>> df
   Texas
b      4
c      7

三、灵活算术方法

方法 描述
add, radd 加法(+)
sub, rsub 减法(-)
div, rdiv 除法(/)
floordiv, floordiv 整除(//)
mul, rmul 乘法(*)
pow, rpow 幂次方(**)

  • 使用示例

    >>> import numpy as np
>>> import pandas as pd
>>> df1 = pd.DataFrame(np.arange(12.).reshape((3, 4)),
                       columns=list('abcd'))
>>> df2 = pd.DataFrame(np.arange(20.).reshape((4, 5)),
                       columns=list('abcde'))
>>> df1
     a    b     c     d
0  0.0  1.0   2.0   3.0
1  4.0  5.0   6.0   7.0
2  8.0  9.0  10.0  11.0
>>> df2
      a     b     c     d     e
0   0.0   1.0   2.0   3.0   4.0
1   5.0   6.0   7.0   8.0   9.0
2  10.0  11.0  12.0  13.0  14.0
3  15.0  16.0  17.0  18.0  19.0

# 在不重叠的位置会出现空值
>>> df1 + df2
      a     b     c     d   e
0   0.0   2.0   4.0   6.0 NaN
1   9.0  11.0  13.0  15.0 NaN
2  18.0  20.0  22.0  24.0 NaN
3   NaN   NaN   NaN   NaN NaN

# 使用填充值的算术方法
>>> df1.add(df2, fill_value=0)
      a     b     c     d     e
0   0.0   2.0   4.0   6.0   4.0
1   9.0  11.0  13.0  15.0   9.0
2  18.0  20.0  22.0  24.0  14.0
3  15.0  16.0  17.0  18.0  19.0

# r开头的方法表示参数是反转的，下面两个语句等价
>>> 1 / df1
       a         b         c         d
0    inf  1.000000  0.500000  0.333333
1  0.250  0.200000  0.166667  0.142857
2  0.125  0.111111  0.100000  0.090909
>>> df1.rdiv(1)
       a         b         c         d
0    inf  1.000000  0.500000  0.333333
1  0.250  0.200000  0.166667  0.142857
2  0.125  0.111111  0.100000  0.090909

# ---DataFrame和Series间的操作--- #
>>> df = pd.DataFrame(np.arange(12.).reshape((4, 3)), columns=list('bde'), index=['Utah', 'Ohio', 'Texas', 'Oregon'])
>>> series = df.iloc[0]

四、排序

DataFrame.sort_values(by, ascending=True, inplace=False)

  • by: 排序键

  • ascending: 排序方式，默认 ascending=True(升序排序)

  • inplace: 修改被调用的对象，而不是生成一个备份， 默认 inplace=False(不修改被调用的对象)

    • 使用示例

      >>> import pandas as pd
>>> df = pd.DataFrame({'a':[0, 1, 0, 1], 
                       'b':[4, 7, -3, 2], 
                       'c':[3, 4, -2, 1]})
>>> df
   a  b  c
0  0  4  3
1  1  7  4
2  0 -3 -2
3  1  2  1

# 单列排序
>>> df.sort_values(by='b')
   a  b  c
2  0 -3 -2
3  1  2  1
0  0  4  3
1  1  7  4

# 多列排序
>>> df.sort_values(by=['a', 'b'])
   a  b  c
2  0 -3 -2
0  0  4  3
3  1  2  1
1  1  7  4

# 降序排序
>>> df.sort_values(by=['a', 'b'], ascending=False)
   a  b  c
1  1  7  4
3  1  2  1
0  0  4  3
2  0 -3 -2

# 修改被调用的对象, 注意应用此方法没有返回值
>>> df.sort_values(by='b', ascending=False,
                   inplace=True)
>>> df
   a  b  c
1  1  7  4
0  0  4  3
3  1  2  1
2  0 -3 -2

六、处理缺失值

1. 过滤缺失值

DataFrame.dropna(axis=0, inplace=False, how=None)

  • axis: 过滤的轴向，默认 axis=0(行)

  • inplace: 修改被调用的对象，而不是生成一个备份， 默认 inplace=False(不修改被调用的对象)

  • how: 删除方式，默认 how=’any’(删除包含缺失值的行或列)

    • 使用示例

      >>> import pandas as pd
>>> from numpy import nan as NA
>>> df1 = pd.DataFrame([[1., 6.5, 3., 2.6], 
                        [1., 6.5, 3., NA], 
                        [1., NA, NA, NA], 
                        [NA, NA, NA, NA], 
                        [NA, 6.5, 3., NA]])
>>> df2 = df1.T
>>> df1
     0    1    2    3
0  1.0  6.5  3.0  2.6
1  1.0  6.5  3.0  NaN
2  1.0  NaN  NaN  NaN
3  NaN  NaN  NaN  NaN
4  NaN  6.5  3.0  NaN
>>> df2
     0    1    2   3    4
0  1.0  1.0  1.0 NaN  NaN
1  6.5  6.5  NaN NaN  6.5
2  3.0  3.0  NaN NaN  3.0
3  2.6  NaN  NaN NaN  NaN

# 删除包含缺失值的行
>>> df1.dropna()
     0    1    2    3
0  1.0  6.5  3.0  2.6

# 删除包含缺失值的列
>>> df2.dropna(axis=1)
     0
0  1.0
1  6.5
2  3.0
3  2.6

# 删除所有值均为空的行
>>> df1.dropna(how='all')
     0    1    2    3
0  1.0  6.5  3.0  2.6
1  1.0  6.5  3.0  NaN
2  1.0  NaN  NaN  NaN
4  NaN  6.5  3.0  NaN

# 删除所有值均为空的列
>>> df2.dropna(axis=1, how='all')
     0    1    2    4
0  1.0  1.0  1.0  NaN
1  6.5  6.5  NaN  6.5
2  3.0  3.0  NaN  3.0
3  2.6  NaN  NaN  NaN

# 修改被调用的对象, 注意应用此方法没有返回值
>>> df1.dropna(inplace=True)
>>> df1
     0    1    2    3
0  1.0  6.5  3.0  2.6
>>> df2.dropna(axis=1, how='all', inplace=True)
>>> df2
     0    1    2    4
0  1.0  1.0  1.0  NaN
1  6.5  6.5  NaN  6.5
2  3.0  3.0  NaN  3.0
3  2.6  NaN  NaN  NaN

2. 补全缺失值

DataFrame.fillna(value, method=None, axis=None, inplace=False, limit=None)

  • value: 标量值或字典型对象用于填充缺失值

  • method: 插值方法

  • axis: 需要填充的轴，默认 axis=0

  • inplace: 修改被调用的对象，而不是生成一个备份， 默认 inplace=False(不修改被调用的对象)

  • limit: 用于填充时最大的填充范围

    >>> import pandas as pd
>>> from numpy import nan as NA
>>> df = pd.DataFrame([[1., 6.5, 4.], [2., NA, NA], [NA, NA, NA], [NA, 2.3, 3.2]])
>>> df
     0    1    2
0  1.0  6.5  4.0
1  2.0  NaN  NaN
2  NaN  NaN  NaN
3  NaN  2.3  3.2

# 使用常值代替缺失值
>>> df.fillna(0)
     0    1    2
0  1.0  6.5  4.0
1  2.0  0.0  0.0
2  0.0  0.0  0.0
3  0.0  2.3  3.2

# 使用字典为不同列设置不同的填充值
>>> df.fillna({1:0.5, 2:1})
     0    1    2
0  1.0  6.5  4.0
1  2.0  0.5  1.0
2  NaN  0.5  1.0
3  NaN  2.3  3.2

# 向后填充
>>> df.fillna(method='ffill')
     0    1    2
0  1.0  6.5  4.0
1  2.0  6.5  4.0
2  2.0  6.5  4.0
3  2.0  2.3  3.2

# 向后填充，最大填充范围为1
>>> df.fillna(method='ffill', limit=1)
     0    1    2
0  1.0  6.5  4.0
1  2.0  6.5  4.0
2  2.0  NaN  NaN
3  NaN  2.3  3.2

# 向前填充
>>> df.fillna(method='bfill')
     0    1    2
0  1.0  6.5  4.0
1  2.0  2.3  3.2
2  NaN  2.3  3.2
3  NaN  2.3  3.2

# 向前填充，最大填充范围为1
>>> df.fillna(method='bfill', limit=1)
     0    1    2
0  1.0  6.5  4.0
1  2.0  NaN  NaN
2  NaN  2.3  3.2
3  NaN  2.3  3.2

# 向右填充
>>> df.fillna(method='ffill', axis=1)
     0    1    2
0  1.0  6.5  4.0
1  2.0  2.0  2.0
2  NaN  NaN  NaN
3  NaN  2.3  3.2

# 向右填充，最大填充范围为1
>>> df.fillna(method='ffill', axis=1, limit=1)
     0    1    2
0  1.0  6.5  4.0
1  2.0  2.0  NaN
2  NaN  NaN  NaN
3  NaN  2.3  3.2

# 向左填充
>>> df.fillna(method='bfill', axis=1)
     0    1    2
0  1.0  6.5  4.0
1  2.0  NaN  NaN
2  NaN  NaN  NaN
3  2.3  2.3  3.2

# 向左填充，最大填充范围为1
>>> df.iloc[2, 2] = 1.5
>>> df
     0    1    2
0  1.0  6.5  4.0
1  2.0  NaN  NaN
2  NaN  NaN  1.5
3  NaN  2.3  3.2
>>> df.fillna(method='bfill', axis=1, limit=1)
     0    1    2
0  1.0  6.5  4.0
1  2.0  NaN  NaN
2  NaN  1.5  1.5
3  2.3  2.3  3.2

# 修改被调用的对象, 注意应用此方法没有返回值
>>> df.fillna(method='bfill', limit=2, inplace=True)
>>> df
     0    1    2
0  1.0  6.5  4.0
1  2.0  2.3  1.5
2  NaN  2.3  1.5
3  NaN  2.3  3.2

七、重命名轴索引

DataFrame.rename(index=None, columns=None, inplace=False)

  • index: 要修改的行索引

  • columns: 要修改的列索引

  • inplace: 修改被调用的对象，而不是生成一个备份， 默认 inplace=False(不修改被调用的对象)

    • 使用示例

      >>> import numpy as np
>>> import pandas as pd
>>> df = pd.DataFrame(np.arange(9).reshape((3, 3)),
                      index=['a', 'b', 'c'], 
                      columns=['Ohio', 'Texas',
                               'California'])
>>> df
   Ohio  Texas  California
a     0      1           2
b     3      4           5
c     6      7           8

# 只修改行索引
>>> df.rename(index={'b': 'd', 'c': 'e'})
   Ohio  Texas  California
a     0      1           2
d     3      4           5
e     6      7           8

# 只修改列索引
>>> df.rename(columns={'Ohio': 'Utah', 
                       'Texas': 'Colorado'})
   Utah  Colorado  California
a     0         1           2
b     3         4           5
c     6         7           8

# 同时修改行索引和列索引
>>> df.rename(index={'b': 'd', 'c': 'e'}, 
              columns={'Ohio': 'Utah', 
                       'Texas': 'Colorado'})
   Utah  Colorado  California
a     0         1           2
d     3         4           5
e     6         7           8

# 修改被调用的对象, 注意应用此方法没有返回值
>>> df.rename(index={'b': 'd', 'c': 'e'}, 
              columns={'Ohio': 'Utah', 
                       'Texas': 'Colorado'},
              inplace=True)
>>> df
   Utah  Colorado  California
a     0         1           2
d     3         4           5
e     6         7           8
